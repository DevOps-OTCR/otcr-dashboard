# Frontend - OTCR Dashboard

React frontend for OTCR dashboard (to be implemented by frontend team).

## Planned Tech Stack
- **Framework**: React 18+
- **Styling**: Tailwind CSS or styled-components
- **State Management**: Redux Toolkit or Zustand
- **HTTP Client**: Axios or Fetch API
- **Authentication**: Clerk integration
- **Build Tool**: Vite or Create React App

## API Integration

The frontend will consume the backend API endpoints:

- `GET /api/users` - Display user list
- `POST /api/users` - Create new user
- `PUT /api/users/[id]` - Update user
- `DELETE /api/users/[id]` - Delete user

## Getting Started

This will be implemented by the frontend team.

## Backend API Base URL

```
http://localhost:3000/api
```

## Authentication

Will integrate with Clerk for user authentication and authorization.

## Development

1. Backend API must be running on port 3000
2. Frontend will run on port 3001 (or configured port)
3. CORS will be configured for local development
